# SIT725 Task 3.2P - Getting Graphical

## Project Name
Events Explorer

## Description
This is a simple web application built using Express.js and Materialize CSS.  
The server provides a GET REST API endpoint and the client consumes that endpoint to display event cards.

## Technologies Used
- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- Materialize CSS

## How to Run
1. Clone the repository
2. Open terminal in the project folder
3. Run `npm install`
4. Run `node server.js`
5. Open `http://localhost:3000`

## API Endpoint
- `GET /api/events`
